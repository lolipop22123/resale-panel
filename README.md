# Project Resale

Этот проект включает в себя:
- **scripts/bot**: Телеграм-бот и вспомогательные скрипты.
- **panel**: Веб-панель (админ-панель/сайт).
- **shared**: Общие модули и логика.
- **config**: Настройки и конфигурация.

## Установка

1. Создайте виртуальное окружение: `python -m venv venv`
2. Активируйте его: `source venv/bin/activate` (macOS/Linux)
3. Установите зависимости: `pip install -r requirements.txt`
4. Настройте `.env` файл на основе `.env.example`.
